# Motion Guard - MEBALCI AI

YOLOv8 tabanlı gerçek zamanlı güvenlik ve hareket izleme uygulaması.

CustomTkinter arayüzü ile kamera görüntüsü üzerinde:

- İnsan / Araç / İkisi tetikleme seçimi
- Dikdörtgen, Daire, Elips ve Serbest çizim
- Alarm sistemi
- PyInstaller ile EXE build

---

## Özellikler

- YOLOv8n gerçek zamanlı nesne tespiti
- Freehand (serbest çizim) alan seçimi
- Elliptical & circular scanning
- Alarm tetikleme sistemi
- Modern dark UI (CustomTkinter)
- Frozen EXE uyumlu Torch runtime hook

---

## Kurulum

pip install -r requirements.txt

---

## Çalıştırma

python src/main.py
